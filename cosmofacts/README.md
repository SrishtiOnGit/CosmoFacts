# 🌌 CosmoFacts

A stunning, high-performance Flutter application that brings the solar system to your fingertips. Explore planets with smooth 3D-parallax animations, detailed statistics, and a sleek dark-matter UI.

---

## 🚀 Features

* **3D Visual Mode:** Swipe through a perspective-transformed carousel of planets.
* **Detailed Exploration:** View gravity, distance from the sun, and unique descriptions for every planet.
* **Hero Animations:** Seamless transitions between the exploration list and planet details.
* **Optimized Assets:** High-quality local images (AVIF/JPG) for a fast, offline experience.

## 🛠️ Installation & Setup

### Prerequisites
* [Flutter SDK](https://docs.flutter.dev/get-started/install) (Stable Channel)
* Android Studio / VS Code
* Dart 3.0+

### Steps
1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/SrishtiOnGit/cosmofacts.git](https://github.com/SrishtiOnGit/cosmofacts.git)
    cd cosmofacts
    ```

2.  **Install dependencies:**
    ```bash
    flutter pub get
    ```

3.  **Run the app:**
    ```bash
    flutter run
    ```
